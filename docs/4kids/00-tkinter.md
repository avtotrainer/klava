# 00 Рђћ tkinter рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

## рЃарЃљ рЃљрЃарЃўрЃА tkinter

`tkinter` рЃљрЃарЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљ, рЃарЃЮрЃЏ Python-рЃўрЃЌ **рЃЊрЃљрЃЋрЃ«рЃљрЃбрЃЮрЃЌ рЃцрЃљрЃюрЃ»рЃљрЃарЃљ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ**.

рЃљрЃЏ рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃерЃў рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ:
- рЃЊрЃљрЃЋрЃгрЃћрЃарЃЮрЃЌ рЃбрЃћрЃЦрЃАрЃбрЃў
- рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮрЃЌ рЃдрЃўрЃџрЃљрЃЎрЃў
- рЃЊрЃљрЃЋрЃљрЃГрЃўрЃарЃЮрЃЌ рЃдрЃўрЃџрЃљрЃЎрЃА
- рЃЋрЃюрЃљрЃ«рЃЮрЃЌ, рЃарЃЮрЃњрЃЮрЃа рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ

рЃљрЃюрЃБ:
­ЪЉЅ **tkinter рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА**

---

## рЃарЃЮрЃњрЃЮрЃа рЃцрЃўрЃЦрЃарЃЮрЃЉрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў

рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў:
- рЃљрЃарЃљрЃцрЃћрЃарЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЌрЃљрЃЋрЃўрЃАрЃўрЃЌ
- рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏрЃљрЃА, рЃарЃљрЃАрЃљрЃф рЃЋрЃБрЃЌрЃ«рЃарЃљрЃЌ
- рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃЕрЃЋрЃћрЃюрЃА рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃА

tkinter рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ:
1. рЃўрЃ«рЃАрЃюрЃћрЃЉрЃљ
2. рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃЕрЃЋрЃћрЃюрЃА рЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃА
3. рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃА

---

## рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»рЃў Рђћ рЃцрЃљрЃюрЃ»рЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃўрЃгрЃДрЃћрЃЉрЃљ рЃцрЃљрЃюрЃ»рЃарЃўрЃЌ.

```python
import tkinter as tk

root = tk.Tk()
root.mainloop()
````

рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃћрЃА рЃЎрЃЮрЃЊрЃў:

* рЃЦрЃЏрЃюрЃўрЃА рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃА
* рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ
* рЃљрЃа рЃљрЃФрЃџрЃћрЃЋрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃљрЃА

рЃЌрЃБ `mainloop()` рЃљрЃа рЃЊрЃљрЃЋрЃгрЃћрЃарЃћрЃЌ Рђћ рЃцрЃљрЃюрЃ»рЃљрЃарЃљ рЃЏрЃљрЃерЃўрЃюрЃЋрЃћ рЃњрЃљрЃЦрЃарЃћрЃЉрЃљ.

---

## рЃцрЃљрЃюрЃ»рЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ

рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў:

```python
root.title("рЃЕрЃћрЃЏрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃцрЃљрЃюрЃ»рЃљрЃарЃљ")
```

рЃћрЃА рЃбрЃћрЃЦрЃАрЃбрЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃцрЃљрЃюрЃ»рЃарЃўрЃА рЃќрЃћрЃЊрЃљ рЃюрЃљрЃгрЃўрЃџрЃерЃў.

---

## рЃарЃљ рЃљрЃарЃўрЃА Widget (рЃЋрЃўрЃ»рЃћрЃбрЃў)

рЃЋрЃўрЃ»рЃћрЃбрЃў рЃљрЃарЃўрЃА **рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃерЃў рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃў рЃюрЃўрЃЋрЃЌрЃў**.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

* рЃбрЃћрЃЦрЃАрЃбрЃў
* рЃдрЃўрЃџрЃљрЃЎрЃў
* рЃДрЃБрЃЌрЃў рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЕрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ

рЃЋрЃўрЃ»рЃћрЃбрЃћрЃЉрЃў рЃљрЃарЃўрЃА рЃцрЃљрЃюрЃ»рЃарЃўрЃА РђърЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃўРђю.

---

## рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ Рђћ Label

рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ `Label`-рЃА.

```python
label = tk.Label(root, text="рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!")
label.pack()
```

рЃћрЃА рЃЎрЃЮрЃЊрЃў:

* рЃЦрЃЏрЃюрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃА
* рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃерЃў

---

## рЃдрЃўрЃџрЃљрЃЎрЃў Рђћ Button

рЃдрЃўрЃџрЃљрЃЎрЃў рЃљрЃарЃўрЃА рЃўрЃА, рЃарЃљрЃАрЃљрЃф рЃЋрЃљрЃГрЃћрЃарЃЌ.

```python
button = tk.Button(root, text="рЃЊрЃљрЃљрЃГрЃўрЃарЃћ")
button.pack()
```

рЃљрЃ«рЃџрЃљ рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃдрЃўрЃџрЃљрЃЎрЃў.

---

## рЃарЃЮрЃњрЃЮрЃа РђърЃћрЃАрЃЏрЃўрЃАРђю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃљ

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА рЃБрЃюрЃЊрЃљ рЃЋрЃБрЃЌрЃ«рЃарЃљрЃЌ:

> рЃарЃЮрЃфрЃљ рЃдрЃўрЃџрЃљрЃЎрЃА рЃЊрЃљрЃљрЃГрЃћрЃарЃћрЃю, рЃарЃљ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮрЃА?

рЃљрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЋрЃгрЃћрЃарЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА:

```python
def say_hello():
    print("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!")
```

рЃЊрЃљ рЃЋрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃЌ рЃдрЃўрЃџрЃљрЃЎрЃЌрЃљрЃю:

```python
button = tk.Button(root, text="рЃЊрЃљрЃљрЃГрЃўрЃарЃћ", command=say_hello)
button.pack()
```

рЃљрЃ«рЃџрЃљ:

* рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃўрЃАрЃљрЃА
* рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў рЃгрЃћрЃарЃА: `рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!`

---

## рЃАрЃарЃБрЃџрЃў рЃърЃљрЃбрЃљрЃарЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ

```python
import tkinter as tk

def greet():
    print("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ, рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃЮ!")

root = tk.Tk()
root.title("рЃЕрЃћрЃЏрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ")

label = tk.Label(root, text="рЃЊрЃљрЃљрЃГрЃўрЃарЃћ рЃдрЃўрЃџрЃљрЃЎрЃА")
label.pack()

button = tk.Button(root, text="рЃЊрЃљрЃљрЃГрЃўрЃарЃћ", command=greet)
button.pack()

root.mainloop()
```

рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃћрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ:

* рЃ«рЃАрЃюрЃўрЃА рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃА
* рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃбрЃћрЃЦрЃАрЃбрЃА
* рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃдрЃўрЃџрЃљрЃЎрЃА
* рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃА рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃљрЃќрЃћ

---

## рЃарЃЮрЃњрЃЮрЃа рЃцрЃўрЃЦрЃарЃЮрЃЉрЃА tkinter (рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ)

tkinter рЃљрЃЏрЃЉрЃЮрЃЉрЃА:

> РђърЃЏрЃћ рЃЊрЃљрЃЋрЃћрЃџрЃЮрЃЊрЃћрЃЉрЃў, рЃАрЃљрЃюрЃљрЃЏ рЃерЃћрЃю рЃарЃљрЃЏрЃћрЃА рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉРђю

рЃЊрЃљ рЃарЃЮрЃфрЃљ:

* рЃдрЃўрЃџрЃљрЃЎрЃА рЃЊрЃљрЃљрЃГрЃћрЃа
* рЃЎрЃџрЃљрЃЋрЃўрЃерЃА рЃЊрЃљрЃљрЃгрЃЋрЃћрЃЉрЃў

рЃўрЃА рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃА.

---

## рЃ«рЃерЃўрЃарЃљрЃЊ рЃЊрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў

рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃўрЃфрЃЮрЃЊрЃюрЃћрЃю:

РЮї рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ:

* рЃћрЃарЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃерЃў рЃЉрЃћрЃЋрЃарЃў `Tk()`
* `mainloop()`-рЃўрЃА рЃЊрЃљрЃЋрЃўрЃгрЃДрЃћрЃЉрЃљ
* рЃЎрЃЮрЃЊрЃўрЃА рЃБрЃгрЃћрЃАрЃарЃўрЃњрЃЮрЃЊ рЃљрЃарЃћрЃЋрЃљ

РюЁ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ:

* рЃюрЃћрЃџрЃљ рЃАрЃгрЃљрЃЋрЃџрЃљ
* рЃърЃљрЃбрЃљрЃарЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў
* рЃЉрЃћрЃЋрЃарЃў рЃфрЃЊрЃљ

---

## рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА tkinter рЃЎрЃљрЃарЃњрЃў рЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА

tkinter рЃЎрЃљрЃарЃњрЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю:

* рЃерЃћрЃЊрЃћрЃњрЃў рЃЏрЃљрЃерЃўрЃюрЃЋрЃћ рЃЕрЃљрЃюрЃА
* рЃЎрЃЮрЃЊрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ
* рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЌрЃљрЃЏрЃљрЃерЃў рЃЊрЃљ рЃАрЃгрЃљрЃЋрЃџрЃљ рЃћрЃарЃЌрЃљрЃЊ
* рЃЋрЃ«рЃћрЃЊрЃљрЃЋрЃЌ, рЃарЃЮрЃњрЃЮрЃа РђърЃфрЃЮрЃфрЃ«рЃџрЃЮрЃЉрЃАРђю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ

---

## рЃърЃљрЃбрЃљрЃарЃљ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃў

1. рЃерЃћрЃфрЃЋрЃљрЃџрЃћ рЃцрЃљрЃюрЃ»рЃарЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў
