# 03 Рђћ рЃърЃљрЃбрЃљрЃарЃљ рЃЌрЃљрЃЏрЃљрЃерЃў рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА: РђърЃЊрЃљрЃГрЃћрЃарЃљ рЃЊрЃарЃЮрЃќрЃћРђю

## рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃўрЃЊрЃћрЃљ

рЃЕрЃЋрЃћрЃю рЃерЃћрЃЋрЃЦрЃЏрЃюрЃўрЃЌ **рЃърЃљрЃбрЃљрЃарЃљ рЃЌрЃљрЃЏрЃљрЃерЃА**, рЃАрЃљрЃЊрЃљрЃф:

- рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃдрЃўрЃџрЃљрЃЎрЃў
- рЃдрЃўрЃџрЃљрЃЎрЃў рЃДрЃЮрЃЋрЃћрЃџ рЃ»рЃћрЃарЃќрЃћ рЃАрЃ«рЃЋрЃљ рЃљрЃЊрЃњрЃўрЃџрЃќрЃћ рЃњрЃљрЃЊрЃљрЃ«рЃбрЃћрЃЉрЃљ
- рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃЏ рЃБрЃюрЃЊрЃљ рЃЏрЃЮрЃљрЃАрЃгрЃарЃЮрЃА рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃљ
- рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЊрЃљрЃўрЃЌрЃЋрЃџрЃўрЃА рЃЦрЃБрЃџрЃћрЃЉрЃА

рЃћрЃА рЃљрЃарЃўрЃА **рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃў рЃЌрЃљрЃЏрЃљрЃерЃў**, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃарЃбрЃўрЃЋрЃў.

---

## рЃарЃљрЃА рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ рЃљрЃЏ рЃЌрЃљрЃЏрЃљрЃерЃерЃў

рЃљрЃЏ рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЉрЃљрЃЋрЃерЃЋрЃў рЃњрЃљрЃўрЃњрЃћрЃЉрЃА:

- рЃарЃЮрЃњрЃЮрЃа рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЊрЃарЃЮрЃќрЃћ
- рЃарЃЮрЃњрЃЮрЃа рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃћрЃЎрЃарЃљрЃюрЃќрЃћ
- рЃарЃЮрЃњрЃЮрЃа рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃЦрЃБрЃџрЃљ
- рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА `after()` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ
- рЃарЃЮрЃњрЃЮрЃа рЃцрЃўрЃЦрЃарЃЮрЃЉрЃА рЃЌрЃљрЃЏрЃљрЃерЃў

---

## рЃюрЃљрЃЉрЃўрЃ»рЃў 1 Рђћ рЃцрЃљрЃюрЃ»рЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

* рЃЋрЃБрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃЌ рЃќрЃЮрЃЏрЃљрЃА

---

## рЃюрЃљрЃЉрЃўрЃ»рЃў 2 Рђћ рЃЦрЃБрЃџрЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў

* рЃњрЃЋрЃўрЃюрЃЊрЃљ рЃдрЃўрЃџрЃљрЃЎрЃў рЃќрЃБрЃАрЃбрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃЋрЃљрЃЊрЃњрЃўрЃџрЃЮрЃЌ

## рЃюрЃљрЃЉрЃўрЃ»рЃў 5 Рђћ рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ

    move_button()

рЃарЃЮрЃфрЃљ рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћ рЃЊрЃљрЃљрЃГрЃћрЃарЃА рЃдрЃўрЃџрЃљрЃЎрЃА:

* рЃЦрЃБрЃџрЃљ рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ
* рЃдрЃўрЃџрЃљрЃЎрЃў рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА рЃљрЃ«рЃљрЃџ рЃљрЃЊрЃњрЃўрЃџрЃќрЃћ

## рЃюрЃљрЃЉрЃўрЃ»рЃў 6 Рђћ рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃљ

## рЃюрЃљрЃЉрЃўрЃ»рЃў 7 Рђћ рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃљ рЃЊрЃарЃЮрЃќрЃћ

рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА:

* рЃЌрЃљрЃЏрЃљрЃерЃў рЃфрЃЮрЃфрЃ«рЃџрЃЮрЃЉрЃА рЃЊрЃарЃЮрЃќрЃћ

## рЃАрЃарЃБрЃџрЃў рЃЌрЃљрЃЏрЃљрЃерЃў (рЃћрЃарЃЌрЃљрЃЊ)

```python
import tkinter as tk
import random

root = tk.Tk()
root.title("рЃЊрЃљрЃГрЃћрЃарЃљ рЃЊрЃарЃЮрЃќрЃћ")
root.geometry("400x300")

score = 0

label = tk.Label(root, text="рЃЦрЃБрЃџрЃљ: 0", font=("Arial", 14))
label.pack()

button = tk.Button(root, text="рЃЊрЃљрЃЏрЃўрЃГрЃўрЃарЃћ!")

def move_button():
    x = random.randint(0, 320)
    y = random.randint(40, 240)
    button.place(x=x, y=y)

def click_button():
    global score
    score += 1
    label.config(text="рЃЦрЃБрЃџрЃљ: " + str(score))
    move_button()

def move_on_time():
    move_button()
    root.after(1000, move_on_time)

button.config(command=click_button)
button.place(x=150, y=120)

move_on_time()
root.mainloop()
```

## рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЌрЃљрЃЏрЃљрЃерЃў (рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ)

1. рЃЌрЃљрЃЏрЃљрЃерЃў рЃўрЃгрЃДрЃћрЃЉрЃљ
2. рЃдрЃўрЃџрЃљрЃЎрЃў рЃ«рЃбрЃћрЃЉрЃљ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ
3. рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћ рЃфрЃЊрЃўрЃџрЃЮрЃЉрЃА рЃЊрЃљрЃГрЃћрЃарЃљрЃА
4. рЃЊрЃљрЃГрЃћрЃарЃўрЃАрЃљрЃА рЃЦрЃБрЃџрЃљ рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ
5. рЃЊрЃарЃЮ рЃњрЃљрЃЊрЃўрЃА Рєњ рЃдрЃўрЃџрЃљрЃЎрЃў рЃўрЃАрЃћрЃЋ рЃњрЃљрЃарЃЉрЃўрЃА

---

## рЃърЃљрЃбрЃљрЃарЃљ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃў рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

1. рЃерЃћрЃфрЃЋрЃљрЃџрЃћ рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў
2. рЃњрЃљрЃљрЃЊрЃўрЃЊрЃћ рЃцрЃљрЃюрЃ»рЃљрЃарЃљ
3. рЃерЃћрЃфрЃЋрЃљрЃџрЃћ рЃЊрЃарЃЮ (1000 Рєњ 700)
4. рЃЦрЃБрЃџрЃљ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА 5-рЃЊрЃљрЃю
5. рЃњрЃљрЃљрЃарЃЌрЃБрЃџрЃћ рЃЌрЃљрЃЏрЃљрЃерЃў (рЃдрЃўрЃџрЃљрЃЎрЃў рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ)

---

## рЃЏрЃљрЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃџрЃўрЃА рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ

рЃћрЃА рЃЌрЃљрЃЏрЃљрЃерЃў рЃљрЃАрЃгрЃљрЃЋрЃџрЃўрЃА:

* рЃЊрЃарЃЮрЃўрЃА рЃфрЃюрЃћрЃЉрЃљрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃерЃў
* рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃћрЃЉрЃќрЃћ рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃљрЃА
* рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА: *рЃЌрЃБ рЃЊрЃљрЃљрЃГрЃўрЃарЃћ Рєњ рЃЦрЃБрЃџрЃљ рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ*
* рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА

рЃћрЃА рЃљрЃарЃўрЃА **рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЌрЃљрЃЏрЃљрЃерЃў**.

## рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ

РђърЃЊрЃљрЃГрЃћрЃарЃљ рЃЊрЃарЃЮрЃќрЃћРђю:

* рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ
* рЃАрЃљрЃ«рЃљрЃџрЃўрЃАрЃЮрЃљ
* рЃАрЃљрЃњрЃљрЃюрЃЏрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃџрЃЮрЃљ

рЃљрЃЦрЃћрЃЊрЃљрЃю рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃўрЃљ:

* РЈ▒ рЃЊрЃарЃЮрЃўрЃА рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃљ
* ­Ъј» рЃЏрЃўрЃќрЃљрЃюрЃерЃў рЃАрЃарЃЮрЃџрЃљ
* ­Ъј« рЃърЃљрЃбрЃљрЃарЃљ рЃарЃћрЃљрЃЦрЃфрЃўрЃўрЃА рЃЌрЃљрЃЏрЃљрЃерЃў

рЃћрЃА рЃљрЃарЃўрЃА **рЃАрЃгрЃЮрЃарЃў рЃњрЃќрЃљ рЃЌрЃљрЃЏрЃљрЃерЃћрЃЉрЃўрЃЌ рЃАрЃгрЃљрЃЋрЃџрЃљрЃЏрЃЊрЃћ**.
