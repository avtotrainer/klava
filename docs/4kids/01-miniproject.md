````md
# 04 — პატარა პროექტი ბავშვებისთვის (Mini Project)

## პროექტის იდეა

ჩვენ შევქმნით **პატარა პროგრამას ფანჯრით**, რომელიც:

- აქვს ღილაკები
- ითვლის, რამდენჯერ დავაჭირეთ ღილაკს
- რეაგირებს ჩვენს მოქმედებაზე

ეს არის **ნამდვილი პროგრამა**, არა უბრალოდ მაგალითი.

## რას ვისწავლით ამ პროექტში

ამ პროექტის შემდეგ ბავშვი შეძლებს:

- ფანჯრის შექმნას
- ტექსტის გამოჩენას ეკრანზე
- ღილაკის გამოყენებას
- ცვლადის გამოყენებას
- პროგრამის „ფიქრის“ გაგებას

## ნაბიჯი 1 — ფანჯრის შექმნა

ვიწყებთ ყოველთვის ფანჯრით.

```python
import tkinter as tk

root = tk.Tk()
root.title("ჩემი პატარა პროგრამა")
````

ეს კოდი:

* ხსნის ფანჯარას
* აძლევს მას სახელს

---

## ნაბიჯი 2 — მთვლელის შექმნა

მთვლელი არის რიცხვი, რომელიც იზრდება.

```python
count = 0
```

ეს ნიშნავს:

> თავიდან რიცხვი არის ნული

---

## ნაბიჯი 3 — ტექსტის ჩვენება

ჩვენ გვინდა ეკრანზე ვნახოთ რიცხვი.

```python
label = tk.Label(root, text="დაჭერების რაოდენობა: 0")
label.pack()
```

---

## ნაბიჯი 4 — რა მოხდება ღილაკის დაჭერისას

ვწერთ ფუნქციას:

```python
def add_one():
    global count
    count = count + 1
    label.config(text="დაჭერების რაოდენობა: " + str(count))
```

რას აკეთებს ეს ფუნქცია:

* ზრდის რიცხვს ერთით
* ცვლის ტექსტს ეკრანზე

---

## ნაბიჯი 5 — ღილაკის დამატება

ახლა ვქმნით ღილაკს:

```python
button.pack()
```

ღილაკი:

* ეძახის ფუნქციას
* აკეთებს იმას, რაც დავუწერეთ

---

## ნაბიჯი 6 — პროგრამის გაშვება

ბოლოს ვწერთ:

```python
root.mainloop()
```

ეს ხაზი:

* აცოცხლებს პროგრამას
* აძლევს მას მუშაობის საშუალებას

---

## სრული პროგრამა (ერთად)

```python
import tkinter as tk

count = 0

def add_one():
    global count
    count = count + 1
    label.config(text="დაჭერების რაოდენობა: " + str(count))

root = tk.Tk()
root.title("დაჭერის მთვლელი")

label = tk.Label(root, text="დაჭერების რაოდენობა: 0")
label.pack()

button = tk.Button(root, text="დააჭირე", command=add_one)
button.pack()

root.mainloop()
```
