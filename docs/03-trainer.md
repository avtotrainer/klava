# 03-trainer.md

## Trainer Рђћ UI-рЃАрЃљ рЃЊрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃўрЃА рЃЊрЃљрЃЏрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃцрЃћрЃюрЃљ

### рЃарЃљрЃА рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА Trainer

**Trainer** рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃБрЃарЃў рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃЮрЃарЃў.
рЃўрЃА **рЃљрЃа рЃўрЃЌрЃЋрЃџрЃўрЃА**, **рЃљрЃа рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃА**, **рЃљрЃа рЃ«рЃљрЃбрЃљрЃЋрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃА** Рђћ рЃЏрЃўрЃАрЃў рЃарЃЮрЃџрЃўрЃљ **рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ**.

рЃЏрЃЮрЃЎрЃџрЃћрЃЊ:

> Trainer рЃљрЃарЃўрЃА РђърЃЏрЃљрЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃћрЃџрЃўРђю, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃА,
> рЃ«рЃЮрЃџрЃЮ рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃЊрЃљ UI рЃљрЃарЃўрЃљрЃю РђърЃерЃћрЃЏрЃАрЃарЃБрЃџрЃћрЃЉрЃџрЃћрЃЉрЃўРђю.

---

### рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА Trainer рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў

рЃЌрЃБ UI рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА рЃћрЃџрЃљрЃърЃљрЃарЃљрЃЎрЃћрЃЉрЃљ:

* рЃЎрЃЮрЃЊрЃў рЃарЃЌрЃБрЃџрЃЊрЃћрЃЉрЃљ
* рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ рЃЌрЃўрЃЌрЃЦрЃЏрЃўрЃА рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃў рЃ«рЃЊрЃћрЃЉрЃљ
* рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃўрЃЦрЃфрЃћрЃЋрЃљ рЃљрЃарЃљрЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃљрЃЊ

Trainer рЃљрЃЏ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљрЃА рЃерЃџрЃўрЃА.

рЃўрЃА рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА:

* рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃЎрЃљрЃцрЃўрЃЮ рЃњрЃљрЃДрЃЮрЃцрЃљрЃА
* рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃА рЃћрЃарЃЌ рЃгрЃћрЃарЃбрЃўрЃџрЃерЃў
* рЃЦрЃфрЃћрЃЋрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљрЃА

---

### Trainer-рЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў

Trainer рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃерЃћрЃЏрЃЊрЃћрЃњрЃќрЃћ:

* рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃўрЃА рЃфрЃўрЃЎрЃџрЃў (рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ / рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃљ / рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ)
* UI рЃЊрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃўрЃА рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ
* рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃћрЃЉрЃўрЃА (keypress, timer, рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ) рЃњрЃљрЃЊрЃљрЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ
* рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў (рЃљрЃЦрЃбрЃўрЃБрЃарЃў / рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў / рЃерЃћрЃЕрЃћрЃарЃћрЃЉрЃБрЃџрЃў)

Trainer **рЃљрЃа**:

* рЃўрЃЌрЃЋрЃџрЃўрЃА рЃЦрЃБрЃџрЃћрЃЉрЃА
* рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃгрЃЮрЃарЃћрЃА
* рЃ«рЃљрЃбрЃљрЃЋрЃА рЃЎрЃџрЃљрЃЋрЃўрЃљрЃбрЃБрЃарЃљрЃА
* рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА

---

### рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃў

рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃюрЃљрЃЎрЃљрЃЊрЃў рЃљрЃАрЃћрЃЌрЃўрЃљ:

```
main.py
   РєЊ
Trainer
   РєЊ
РћїРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћљ     РћїРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћљ
Рћѓ   Logic   Рћѓ РЄё   Рћѓ     UI    Рћѓ
РћћРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћў     РћћРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћў
```

Trainer рЃљрЃарЃўрЃА рЃћрЃарЃЌрЃљрЃЊрЃћрЃарЃЌрЃў рЃцрЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф:

* рЃ«рЃћрЃЊрЃљрЃЋрЃА UI-рЃА
* рЃ«рЃћрЃЊрЃљрЃЋрЃА Logic-рЃА
* рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃА рЃЏрЃљрЃЌ рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃљрЃА

---

### Trainer рЃЊрЃљ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃћрЃЉрЃў (State)

рЃЎрЃљрЃарЃњ Trainer-рЃА рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃЦрЃЋрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃћрЃЉрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

* `training_active`
* `exercise_running`
* `finished`
* `timer_started`

рЃћрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю:

* рЃЉрЃљрЃЋрЃерЃЋрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃљрЃГрЃўрЃарЃЮрЃА рЃЎрЃџрЃљрЃЋрЃўрЃерЃА рЃљрЃарЃљрЃАрЃгрЃЮрЃа рЃЊрЃарЃЮрЃА
* UI рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃБрЃЎрЃЋрЃћ рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃўрЃДрЃЮрЃА
* рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃдрЃљрЃа рЃўрЃдрЃћрЃЉрЃЊрЃћрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃљрЃА

Trainer рЃгрЃДрЃЋрЃћрЃбрЃА:

> РђърЃљрЃ«рЃџрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ?Рђю
> РђърЃљрЃ«рЃџрЃљ рЃБрЃюрЃЊрЃљ рЃўрЃњрЃюрЃЮрЃарЃўрЃарЃћрЃЉрЃљ?Рђю
> РђърЃљрЃ«рЃџрЃљ рЃЊрЃљрЃЋрЃљрЃАрЃарЃБрЃџрЃЮрЃЌ?Рђю

---

### рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ (Event Flow)

рЃбрЃўрЃърЃБрЃарЃў рЃЎрЃџрЃљрЃЋрЃўрЃерЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃўрЃА рЃАрЃфрЃћрЃюрЃљрЃарЃў:

1. UI рЃўрЃдрЃћрЃЉрЃА keypress-рЃА
2. Trainer рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљрЃА
3. Trainer рЃњрЃљрЃЊрЃљрЃАрЃфрЃћрЃЏрЃА рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА
4. рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃерЃћрЃЊрЃћрЃњрЃА (рЃАрЃгрЃЮрЃарЃў / рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў)
5. Trainer:

   * рЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃА UI-рЃА
   * рЃљрЃю рЃўрЃгрЃДрЃћрЃЉрЃА/рЃљрЃЕрЃћрЃарЃћрЃЉрЃА рЃбрЃљрЃўрЃЏрЃћрЃарЃА
   * рЃљрЃю рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮрЃА

рЃџрЃЮрЃњрЃўрЃЎрЃљ **рЃљрЃа рЃўрЃфрЃўрЃА**, рЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ.
UI **рЃљрЃа рЃўрЃфрЃўрЃА**, рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃАрЃгрЃЮрЃарЃў рЃљрЃю рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў.

---

### рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃћрЃА рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃАрЃгрЃЮрЃарЃў рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃљрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ:

* рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃЦрЃфрЃћрЃЋрЃљ
* рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃќрЃћ рЃЏрЃЎрЃљрЃцрЃўрЃЮ рЃарЃћрЃљрЃЦрЃфрЃўрЃљ
* рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃЉрЃљрЃњрЃћрЃЉрЃўрЃА рЃљрЃарЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљ

Trainer рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА:

* рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃЮрЃЉрЃљрЃА
* рЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА
* рЃЦрЃфрЃћрЃЋрЃўрЃА рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃБрЃџрЃЮрЃЉрЃљрЃА

рЃћрЃА рЃљрЃарЃўрЃА **рЃАрЃљрЃњрЃљрЃюрЃЏрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃџрЃЮ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ**, рЃљрЃарЃљ рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў.

### Trainer-рЃерЃў рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ:
