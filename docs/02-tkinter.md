# 02 — tkinter

## შესავალი

`tkinter` არის Python-ის **სტანდარტული გრაფიკული ბიბლიოთეკა**  
(Graphical User Interface — GUI).

მისი დანიშნულებაა პროგრამისთვის **ფანჯრის, ღილაკების, ტექსტის, მენიუს და სხვა ვიზუალური ელემენტების** შექმნა.

თუ Python უკვე გაქვს დაყენებული, `tkinter` ჩვეულებრივ უკვე ხელმისაწვდომია.

---

## რას ნიშნავს ბიბლიოთეკა Python-ში

Python-ში **ბიბლიოთეკა** არის კოდის ერთობლიობა, რომელიც:
- დაწერილია ერთხელ
- გამოიყენება ბევრ პროგრამაში
- ამარტივებს რთული ამოცანების შესრულებას

ბიბლიოთეკა გვაძლევს **მზად ფუნქციებსა და კლასებს**, რათა:
- თავიდან არ ვწეროთ ყველაფერი
- კონცენტრირდეთ ამოცანაზე და არა ტექნიკურ დეტალებზე

def on_click():
    print("ღილაკი დააჭირეს")

btn = tk.Button(root, text="დააჭირე", command=on_click)
btn.pack()

root.mainloop()
